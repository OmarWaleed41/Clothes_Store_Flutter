# Clothes_Store_Flutter

Key components:
- `Product` model
- `HomePage` (product listing & filtering)
- `DetailPage` (product details)
- `CartPage` (cart management)
- `FavoritePage` (favorites list)

---

## Core Concepts Demonstrated

- `StatelessWidget` vs `StatefulWidget`
- List filtering and dynamic UI updates
- Passing data between screens
- Callback functions between widgets
- Basic Flutter Material components:
  - `Scaffold`
  - `AppBar`
  - `ListView`
  - `Card`
  - `SnackBar`
